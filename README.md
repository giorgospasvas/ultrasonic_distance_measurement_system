#  Ultrasonic Distance Measurement System

A Distance measurement system using an Arduino Nano, an ultrasonic module, and a TM1637 display.

## 🛠️ Components & Hardware
- **Microcontroller:** Arduino Nano
- **Sensors:**  HC-SR04 Ultrasonic Sensor
- **Display:** TM1637
- **Input:** Pushbutton 6mm , flipswitch
- **Power:** Battery 9 V
